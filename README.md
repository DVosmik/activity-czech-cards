## Refreshing on iPhone

To make sure you're seeing the latest version on iPhone Safari:

1. Open the site in browser
2. Tap the **URL bar** at the top/bottom
3. Pull down to refresh, or tap the **reload ↻** button
4. If the version shown at the bottom of the page hasn't changed, clear the cache:
   **Settings → Safari → Clear History and Website Data**
